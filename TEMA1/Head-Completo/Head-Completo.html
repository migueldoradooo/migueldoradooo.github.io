IES

Git es un sistema de control de versiones distribuido de código abierto que se utiliza para rastrear cambios en el código fuente de un proyecto, permitiendo a los desarrolladores colaborar en él de forma eficiente. Permite crear "instantáneas" (commits) del proyecto para guardar su estado en diferentes momentos, fusionar cambios y volver a versiones anteriores si es necesario.

¿Que es SEO?

SEO significa "Search Engine Optimization" (Optimización para Motores de Búsqueda) y se refiere al conjunto de técnicas informáticas para mejorar la visibilidad y el posicionamiento de un sitio web en los resultados de buscadores como Google. El objetivo principal es atraer tráfico orgánico de calidad a través de estrategias como el uso de palabras clave, la optimización del contenido, la mejora técnica del sitio web y la obtención de enlaces de calidad (backlinks).

OpenGraph

Open Graph es un protocolo informático que permite a los desarrolladores controlar cómo se muestra el contenido de una página web cuando se comparte en redes sociales. Utiliza metaetiquetas (etiquetas en el código HTML que no son visibles para el usuario) para especificar datos como el título, la descripción y la imagen principal que se verá en la vista previa de la publicación. Esto ayuda a las plataformas sociales a interpretar correctamente el contenido y presentarlo de manera más atractiva y coherente.
