HTML, siglas de HyperText Markup Language (Lenguaje de Marcas de Hipertexto), es el lenguaje estándar utilizado para crear y estructurar páginas web

CSS, o Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje de diseño gráfico fundamental para el desarrollo web que se utiliza para estilizar elementos escritos en lenguajes de marcado como HTML o XML
